# Задание 1: Знакомство с Docker Compose

Docker Compose — это инструмент для управления многоконтейнерными приложениями. Он позволяет запускать целые связки сервисов одной командой, избавляя от ручной настройки сети и зависимостей. Для меня это способ поддерживать чистоту в системе: проекты живут в своих изолированных средах, не конфликтуют друг с другом и разворачиваются за секунды, что экономит массу времени при разработке.
